Mistype

Mistype is a set of styles that helps you to design a new site. It's a simple CSS prototyping framework, put in a 2.5 kB file.

Download

Downloading and using Mistype is pretty easy.

Browser support

  • Chrome (latest)
  • Firefox (latest)
  • Opera (latest)
  • Edge (latest)
  • Safari 7+
  • Internet Explorer 10+
